## 🎯 Common Workflows

### Historical Impact
```
1. Quick Actions → Presets dropdown
2. Select "Chicxulub (Dinosaurs)"
3. Click Start
4. Watch extinction event
```

### Real Asteroid Analysis
```
1. Quick Actions → NASA Data
2. Select Apophis
3. Click Analyze Impact
4. View risk assessment
```

### Custom Impact Location
```
1. Quick Actions → Map Impact
2. Click your city on map
3. Adjust size/velocity
4. See damage zones
```

### Compare Multiple Asteroids
```
1. Left Panel → Custom Tab
2. Create first asteroid
3. Click "+ Add Asteroid"
4. Repeat and compare
```

## 📊 Understanding the Numbers

| Event | Size | Energy | Effect |
|-------|------|--------|--------|
| Chelyabinsk | 20m | 0.5 MT | 1,500 injured |
| Tunguska | 60m | 10 MT | 2,000 km² forest |
| Barringer | 50m | 10 MT | Arizona crater |
| Chicxulub | 10 km | 100M MT | Extinction |

**Reference:**
- 1 MT = 1 million tons TNT
- Hiroshima ≈ 0.015 MT
- Largest nuke ≈ 50 MT
